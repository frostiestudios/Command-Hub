MKDocs Commands

These are commands that are ran directly via. the CLI or Terminal You may also use the GUI program but that may not be working * mkdocs new [dir-name] - Create a new project. * mkdocs serve - Start the live-reloading docs server. * mkdocs build - Build the documentation site. * mkdocs -h - Print help message and exit.
